## 第三章：判断语句

### 比较运算符

| 语法 | 描述         |
| ---- | ------------ |
| ==   | 是否相等     |
| !=   | 是否不相等   |
| >    | 是否大于     |
| >=   | 是否大于等于 |
| <    | 是否小于     |
| <=   | 是否小于等于 |

### if elif else

```python
num1 = 10

if num1 > 10:
  print('大于10')
elif num1 < 10:
  print('小于10')
else:
  print('等于10')
```
